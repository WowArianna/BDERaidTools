# BDERaidTools
A lightweight utility addon designed as a replacement for WeakAura.

## Installation
1. Download the latest `.zip` from GitHub:  
   https://github.com/WowArianna/BDERaidTools/releases/
2. Extract **BDERaidTools.zip** into:  
   `World of Warcraft/_retail_/Interface/AddOns/`

## Configuration
All features can be enabled or disabled through the World of Warcraft options menu.

## Jesus
The Jesus module reminds you not to give up and to avoid releasing your spirit, as a resurrection may still be possible.  

<img width="312" height="312" alt="grafik" src="https://github.com/user-attachments/assets/84d3b22e-1948-41fb-ae3b-5b1142db02c2" />

You can test the module using the following console commands:
- `/bdejesus`
- `/bdejesus show`
- `/bdejesus hide`

## M+ Tooltip
This feature adds Mythic+ information retrieved from the official WoW API. The data appears when hovering over a player or unit frame.  
The tooltip is only displayed when:
- you are **not in combat**, and
- you are **inside the same instance** as the target.
